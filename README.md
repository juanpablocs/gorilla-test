# Gorilla Test App

clone repository and install dependencies

```
npm install --force
```

It is necessary to install the dependencies in a forced way, because this library (styled-components) is incompatible with the latest version of react.

## run on ios

```
npm run ios
```

## run on android

```
npm run android
```
